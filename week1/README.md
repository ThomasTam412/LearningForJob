🕳️ 踩過的坑（重要！）
坑	教訓
viewprot 拼錯	細節敏感度，1 個字母毀掉手機版

平板斷點重疊（768 同時出現在兩段）	邊界值要明確分開
直接寫 100% - 20px	CSS 數學運算必須包 calc()
picsum.photos 掛掉	不是所有 bug 都是自己的問題，要會驗證

💡 學習心得（自我反思點）
「看懂 ≠ 會寫」
Day5 第一次手打才發現很多東西記得不牢，這就是練習的價值。

「先求有再求好」
不要一開始就追求完美，先把東西做出來再優化。

「複製 vs 手打的判斷」
舊檔案可複製當基礎，新學的必須手打。

「不准依賴 AI 補完」
關掉 Copilot/Emmet 後，學習速度反而更快。

「練習日比學習日更重要」
下週開始改用「學2天練1天」的節奏。
