### 第 5～6 週：PHP 基礎

**安裝 XAMPP，讓 PHP 能在你的電腦上跑。**

**第 5 週：PHP 語法**

```php
<?php
// 變數
$name = "Tom";

// 字串
echo "Hello " . $name;

// 條件
if ($age >= 18) { echo "成年"; }

// 迴圈
for ($i = 0; $i < 10; $i++) { echo $i; }

// 陣列
$arr = [1, 2, 3];
$arr[] = 4; // push
foreach ($arr as $item) { echo $item; }

// 關聯陣列（類似物件）
$user = ["name" => "Tom", "age" => 20];
echo $user["name"];

// 函數
function add($a, $b) { return $a + $b; }
?>
```

**每天練習：**

**第 6 週：PHP + 表單 + 文件**

### 第 7～8 週：MySQL 基礎

**學 SQL，用 phpMyAdmin（XAMPP 自帶）練習。**

**第 7 週：SQL 語法**

```sql
-- 建表
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(50),
    email VARCHAR(100),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 插入
INSERT INTO users (name, email) VALUES ('Tom', 'tom@example.com');

-- 查詢
SELECT * FROM users;
SELECT * FROM users WHERE name = 'Tom';
SELECT * FROM users ORDER BY created_at DESC;
SELECT * FROM users LIMIT 10;

-- 更新
UPDATE users SET name = 'Jerry' WHERE id = 1;

-- 刪除
DELETE FROM users WHERE id = 1;
```

**每天練習：**

```text
Day 1：在 phpMyAdmin 建一張 users 表
Day 2：插入 10 筆資料
Day 3：各種 SELECT 查詢練習
Day 4：UPDATE 和 DELETE 練習
Day 5：建第二張表 posts，練習兩張表的關係
Day 6：JOIN 查詢（查用戶和他的文章）
Day 7：GROUP BY（統計每個用戶有幾篇文章）
```

**第 8 週：PHP + MySQL**

```php
// PDO 連接
$pdo = new PDO('mysql:host=localhost;dbname=mydb', 'root', '');

// 查詢
$stmt = $pdo->query("SELECT * FROM users");
$users = $stmt->fetchAll();

// 插入（防 SQL 注入）
$stmt = $pdo->prepare("INSERT INTO users (name, email) VALUES (?, ?)");
$stmt->execute([$name, $email]);
```

**每天練習：**

```text
Day 1：PHP 連接 MySQL，查詢所有用戶並顯示
Day 2：做一個新增用戶的表單（寫入資料庫）
Day 3：做刪除功能
```
